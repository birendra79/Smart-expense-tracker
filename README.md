# 💸 Smart Expense Tracker

An AI-powered expense tracker that automatically categorizes your expenses, sends real-time budget alerts, and predicts future spending — built as a hobby project to explore machine learning and full-stack development.

---

## 🛠️ Built With

- **HTML / CSS / JavaScript** — Frontend
- **Node.js + Express** — Backend
- **Google Apps Script** — Spreadsheet automation & data handling
- **Machine Learning** — Spending prediction model

---

## 🚀 Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/birendra79/Smart-expense-tracker.git
cd Smart-expense-tracker
```

### 2. Setup Backend
```bash
cd backend
npm install
```

Create a `.env` file in the `backend` folder:
```env
PORT=5000
```

```bash
npm start
```

### 3. Open Frontend
```bash
cd frontend
# Open index.html in your browser
# or use Live Server in VS Code
```

### 4. Apps Script (optional)
The `apps-script` folder contains Google Apps Script code for Google Sheets integration. Deploy it from [script.google.com](https://script.google.com) if you want spreadsheet-based tracking.

---

## ✨ Features

- Auto-categorizes expenses using AI
- Real-time budget alerts when you overspend
- Predicts future spending based on your history
- Google Sheets integration via Apps Script
- Clean, simple dashboard UI
